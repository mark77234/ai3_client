1. **뉴스 감성 판단**  
   - 투자자 의견(긍정 vs 부정) 반영 → **20%**  
2. **애널리스트 평가**  
   - 전문 애널리스트의 뉴스 영향력 점수 → **40%**  
3. **기업 영향도 분석**  
   - 재무 지표, 시장 반응 등 종합 평점 → **40%**  

> 위 세 요소를 가중합하여 최종 AI 점수(`0~100`)를 산출합니다.  
> 점수별 투자 가이드라인(매수·홀딩·매도)을 제공해, 사용자들이  
> 한눈에 이해하고 의사결정할 수 있도록 설계했습니다.
